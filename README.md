# 📖 Overview

The **AI Healthcare Anomaly Detection System** is a Machine Learning-powered web application designed to identify unusual patterns in healthcare data and assist medical professionals in early diagnosis and risk assessment.

The system analyzes patient information and predicts whether the provided health parameters indicate normal or abnormal conditions. It provides an intuitive dashboard for monitoring and visualization, making healthcare analysis faster and more efficient.

# 🚀 Installation

## Clone the Repository

```bash
git clone https://github.com/HarishBirla/AI-Healthcare-Anomaly-Detection-System.git

cd AI-Healthcare-Anomaly-Detection-System
```

---

## Create Virtual Environment

### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

### Linux / macOS

```bash
python3 -m venv venv
source venv/bin/activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Run the Project

```bash
python app.py
```

The application will start on:

```
http://127.0.0.1:5000
```

Open the above URL in your browser.
